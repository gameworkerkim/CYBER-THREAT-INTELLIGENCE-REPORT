### KelpDAO LayerZero 브리지 해킹 — 오프체인 검증 인프라의 단일 장애점

**1-of-1 DVN, RPC 노드 포이즈닝, 그리고 DeFi 전반으로 번진 시스템 리스크**

2026년 4월 18일, 북한 연계 Lazarus Group의 하위 조직 TraderTraitor가 KelpDAO의 LayerZero 브리지 인프라를 공격해 **116,500 rsETH(약 2억 9,200만 달러)** 를 탈취했다. 스마트 컨트랙트 버그가 아닌 오프체인 검증 인프라의 1-of-1 DVN 단일 장애점과 RPC 노드 포이즈닝을 정밀하게 파고든 신종 수법으로, **2026년 최대 DeFi 해킹**이자 북한이 단 2건의 해킹으로 2026년 전 세계 암호화폐 해킹 피해의 76%를 차지하게 만든 사건이다.
